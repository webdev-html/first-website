what should you learn for frontend?


first, you will need to learn html, css, and js(javascript), these are the 3 fundemental laungages in frontend web devlopment, and you will use them all the time in frontend development.

now, on to the frameworks/extensions.

something I and many other js programmers recommend is typescript, it is a super set of javascript, and to summarize it, it is just javascript, but better(you can read the full documentation for typescript here)

(node.js is technically a backend framework, but it is heavily tied with frontend)
you should also learn node.js(or just have it installed) because what node.js does is that it makes it so that you can use javascript for backend, not just frontend, and it makes projects more efficent.

another reason to have node.js installed in your path is because it comes with something called npm(node package manager), with this, you can download most major frameworks/extensions(like react and typescript) through it's command line.

other frameworks you should learn could be angular or react(react is technicaly a library), i just haven't learned angular or react or got the gist of it, so i can't give my personal opinion.
